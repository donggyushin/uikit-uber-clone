# uikit-uber-clone

iOS15 UIKit 을 활용해서 Uber 앱을 클론한다. 

## 무엇을 배울것인가

- GeoFire을 이용해 Geolocation 기능 사용
- Firebase를 이용해 serverless backend 구현
- UI 테크닉
- CoreLocation 과 MapKit
- UIKit을 이용한 Animation
- 유저 인증 기능
- realtime database를 이용한 유저들간 실시간 상호작용
